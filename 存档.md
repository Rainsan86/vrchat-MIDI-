# MIDI Show 项目存档

> 存档日期：2026-07-05 15:04

## 项目概述

MIDI Show 是一个基于 Python 的 MIDI 播放器，专为 VRChat 虚拟直播设计。使用 `python-rtmidi` 驱动虚拟 MIDI 端口，配合 VRChat 中的 MIDI 化乐器（如 Fractal Skeleton 的 MIDI 鼓、Memes 的 MIDI 吉他等），实现从 MIDI 文件到 VRChat 演奏的完整管线。

## 项目结构

```
midi_show/
├── __init__.py          # 版本信息
├── engine.py            # 核心引擎（播放、循环、节奏控制、冲突检查）
├── i18n.py              # 国际化支持 (英文/中文)
├── library.py           # MIDI 曲库管理
├── midi_parser.py       # MIDI 文件解析器
├── outputs.py           # 输出/导出功能
├── settings.py          # 设置管理器
└── ui.py               # Tkinter 图形界面

main.py                  # 入口文件
requirements.txt         # Python 依赖
start.bat               # Windows 启动脚本
start.ps1               # PowerShell 启动脚本
midi_show_config.json            # 用户配置（不提交到 git）
midi_show_config.sample.json     # 配置模板（提交到 git）
midi_show_library.json           # 曲库数据（不提交到 git）
midi_show_library.sample.json    # 曲库模板（提交到 git）
backups/                         # JSON 数据库备份
```

## 核心技术栈

- **语言**: Python 3.10+
- **MIDI 驱动**: python-rtmidi (通过 loopMIDI 创建虚拟端口)
- **UI**: Tkinter (标准库)
- **存储**: JSON 文件（配置 + 曲库）

## 启动方式

```bash
python main.py
# 或双击 start.bat
```

需要先运行 loopMIDI 创建虚拟端口。
